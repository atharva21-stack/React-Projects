## Run Locally

Clone the project

```bash
  git clone https://github.com/atharva21-stack/React-Projects
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
